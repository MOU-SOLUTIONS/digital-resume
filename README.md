# Digital Resume

A production-grade, multilingual digital resume and portfolio website built with modern Angular architecture.

## Overview
This project showcases professional experience, technical skills, projects, certifications, and community engagement in a highly optimized, accessible, and SEO-friendly single-page application.

## Tech Stack
- Angular (Standalone Components, OnPush)
- TypeScript (Strict Mode)
- SCSS
- RxJS

## Key Features
- Internationalization (EN, FR, AR, ES) with RTL support
- Light / Dark theme system
- SEO optimization (Schema.org, Open Graph, meta tags)
- Accessibility (ARIA, keyboard navigation)
- Performance-first architecture
- Responsive, mobile-first design
